    Interface Robtimus\Obfuscation\ObfuscatorPrefix

    A prefix of a specific length that uses a specific obfuscator. It can be used to create combined obfuscators that obfuscate text for the part up to the length of this prefix using the prefix' obfuscator, then the rest with another.

    Returns an obfuscator that first uses the source of this object for the length of this prefix, then another obfuscator. If the length of text to obfuscate is smaller than or equal to the length of this prefix, the other obfuscator will be skipped.

    The returned obfuscator is immutable if both the source of this object and the other obfuscator are.

    Parameters

    $other  

    The other obfuscator to use for text after the length of this prefix has been exceeded.

    Returns

    An obfuscator that combines the two obfuscators.
